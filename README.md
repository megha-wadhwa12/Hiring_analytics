### Hiring Pipeline & Candidate Funnel Analytics

A CLI-based hiring analytics system built using **pure Python and SQLite**.
The system simulates a real-world hiring pipeline and provides SQL-based
analytics for recruitment insights.

#### Analytics Reports
- Stage-wise funnel drop-off
- Hiring success ratio
- Interviewer performance
- Average time to hire
- Top rejection reasons
